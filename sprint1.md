User stories
As a student, I want to create a task with a deadline and priority so that others understand its urgency
As a student, I want to edit my tasks so that I can fix mistakes or update them when needed.
As a student, I want to mark myself as the owner of a task so that everyone clearly knows I am responsible for completing it and tracking its progress.
As a student, I want to delete a task, so that I can remove completed or unnecessary tasks
As a student, I want to see the status of a task so that I can track its progress.
