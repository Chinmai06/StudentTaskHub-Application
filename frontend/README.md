# Student Task Hub Frontend

This is the frontend-only React version of the Student Task Hub project.

## Run locally

```bash
npm install
npm run dev
```

## Included files
- React source files in `src/`
- App entry files: `index.html`, `src/main.jsx`, `src/App.jsx`
- Vite config and package file

## Not included
- Cypress tests
- Unit test files
- Backend files
